欢迎来到 Flask 世界
===================

.. image:: _static/flask-logo.png
    :alt: Flask: web development, one drop at a time
    :align: center
    :target: https://palletsprojects.com/p/flask/

这就是 Flask 文档。从安装指导 :ref:`installation` 开始，
安装完后来看一下 :ref:`quickstart` 文档，你会得到一个概括了解。
更详细的内容参考 :ref:`tutorial` 我们的教程，该教程会让你了解
如何建立一个麻雀虽小五脏俱全的 Flask 应用。共同采用的模式都会描述
在 :ref:`patterns` 模式文档部分。接下来的文档描述了 Flask 中
每个组件的细节，要想全部掌握请参考 :ref:`api` 这部分文档内容。

Flask 依据 `Jinja`_ 模版引擎和 `Werkzeug`_ WSGI 工具集。
它们的文档可以在如下网址找到：

- `Jinja documentation <http://jinja.pocoo.org/docs>`_
- `Werkzeug documentation <http://werkzeug.pocoo.org/docs>`_

.. _Jinja: https://www.palletsprojects.com/p/jinja/
.. _Werkzeug: https://www.palletsprojects.com/p/werkzeug/
