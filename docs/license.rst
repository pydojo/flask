协议
=======

Flask 协议位于一种三条款 BSD 协议之下。基本意思就是：
做你想做的，只要 Flask 贴附的版权一直存在，所有条件都不能修改并且要表明免责声明。
进一步来说，没有本软件作者们的书面授权，你不能使用作者的名字来宣传本软件的衍生版本。

完整的协议文字可以在如下参考文档中找到（:ref:`flask-license`）。
对于文档与艺术作品拥有不同的协议。
